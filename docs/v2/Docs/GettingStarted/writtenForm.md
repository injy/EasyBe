# 书写规范

* 富文本编辑器建议：TinyMCE
* MD编辑器建议：Markdown

|**类型**|**TinyMCE**|**Markdown**|
|:-----:|:-----:|:-----:|
|**一级标题**|```<h1>```|```#```|
|**二级标题**|```<h2>```|```##```|
|**引用**|```引用```|```>```|
|**重要/警告引用**|```!>```|```!>```|
|**普通/信息引用**|```?>```|```?>```|
|**行内代码**|```行内代码```| ```|
|**代码折叠**| | ```<details><summary> 代码 </summary>// 空一行代码块内容// 空一行</details>```|
