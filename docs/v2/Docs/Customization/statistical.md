#### 博皮配置

```js
window.cnblogsConfig = {
    umami: {
        url: 'https://umami.wangyangyang.vip',
        shareId: '0uuvzPh5',
    },
}
```

![博皮配置](https://cdn.jsdelivr.net/gh/wangyang0210/pic/imgs/project/easybe/6.gif)

#### 集成效果

| 字段      | 解释                             |
| --------- | -------------------------------- |
| Online    | 当前在线人数                     |
| Today     | 今天的数据 pv/uv/平均访问时长(s) |
| Yesterday | 昨天的数据 pv/uv/平均访问时长(s) |
